# Defense of the Animals

你将要面对的是一群穷凶极恶的怪兽，它们正在虎视眈眈地盯着你的小动物们。赶快召唤强大的守卫来保护你的动物朋友吧！仙人球、灵魂激流和连环闪电会把敌人打得落花流水，当然，如果你感觉不够劲，还可以召唤出混沌陨石、传送门等等强有力的道具(当然都是要花钱的)。发挥你的才智和勇气，把这群怪兽打得片甲不留！
